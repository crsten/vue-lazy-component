# vue-make-lazy

## Docs

```html
<MakeLazy>
  <!-- Your content that will be lazyloaded -->
</MakeLazy>
```

```js
// Import and register
import MakeLazy from 'vue-make-lazy'

// Vue 2
Vue.component('MakeLazy', MakeLazy)
// Vue 3
app.component('MakeLazy', MakeLazy)
```
