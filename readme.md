# vue-lazy-component

## Docs

```html
<LazyLoader>
  <!-- Your content that will be lazyloaded -->
</LazyLoader>
```

```js
// Import and register
import LazyLoader from 'vue-lazy-component'

// Vue 2
Vue.component('LazyLoader', LazyLoader)
// Vue 3
app.component('LazyLoader', LazyLoader)
```
